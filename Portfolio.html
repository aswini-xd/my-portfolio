Friend Request Card

    This project is a dynamic friend request card built using HTML, CSS, and JavaScript. The card features a profile image, a name, and a status indicator that toggles between "STRANGER" and "FRIENDS" upon user interaction.

    Key functionalities include:

  • Dynamic Status Update: Clicking the "ADD FRIEND" button changes the status to "FRIENDS", updates the text color to green, and modifies the button label to "Remove". Clicking again reverts the status.
  • Modern UI Design: Styled with CSS to create a visually appealing card layout with smooth transitions. Interactive User Experience: Uses JavaScript event handling to provide real-time updates.
  • Dynamic Status Update: Clicking the "ADD FRIEND" button changes the status to "FRIENDS", updates the text color to green, and modifies the button label to "Remove". Clicking again reverts the status.
This project showcases front-end development skills, focusing on user interaction and UI design.

Heart Animation on Double Click

    This project features an interactive heart animation effect implemented using HTML, CSS, and JavaScript. The animation is triggered when the user double-clicks on an image, creating a visually appealing interaction.

    Key Features:

  • Smooth Heart Animation: A heart icon appears and scales up on a double-click event, then gradually fades away for a seamless effect.
  • CSS Transitions & Transformations: Uses transform and opacity transitions to achieve a smooth scaling effect.
  • User Engagement: Enhances user experience with an interactive and visually appealing reaction.
  • Lightweight & Responsive: Efficiently designed to work on various screen sizes without performance issues.
This project demonstrates expertise in front-end development, particularly in creating engaging UI animations and event-driven interactions.
